## 1. 风险分级标准

- **严重度（Severity）**：高 / 中 / 低
- **概率（Likelihood）**：高 / 中 / 低
- **优先级（Priority）**：
  - P0：高严重 + 高/中概率
  - P1：高严重 + 低概率，或中严重 + 高概率
  - P2：中严重 + 中/低概率，或低严重 + 高概率
  - P3：低严重 + 中/低概率

## 4. 关闭准则

风险关闭必须满足以下条件：

1. 已执行对应缓释措施；
2. 有可复现证据（命令输出、日志、截图或导出文件）；
3. 在周验收或 UAT 记录中可追溯到风险 ID；
4. 由责任人与审核人共同确认。
