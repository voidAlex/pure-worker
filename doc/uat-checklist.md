## 使用说明

本文档用于 UAT（用户验收测试）阶段的质量门禁检查，涵盖功能、质量、安全三个维度。

**填写规范：**
- 每个检查项必须勾选 `- [x]` 或保留待检 `- [ ]`
- "实际结果"列填写具体测试数据或观察结论
- "验收人"列填写执行验收的人员姓名
- "日期"列填写验收完成的日期（YYYY-MM-DD）
- 所有检查项必须通过（勾选）方可进入发布阶段

**验收流程：**
1. 功能验收：按 §9.1 逐项验证端到端闭环
2. 质量验收：按 §9.2 执行性能与稳定性测试
3. 安全验收：按 §9.3 执行安全合规检查
4. 签字确认：所有检查项通过后，相关责任人签字

| - [ ] 关键任务成功率 | >= 95%（统计周期：UAT 全程） | | | | |

**关键任务定义：**
- 批量作业结构化任务
- 批量期末评语生成任务
- 家长沟通文案生成任务
- 错题重组与练习生成任务

**测试方法：**
1. 执行 N 次关键任务（N >= 100）
2. 统计成功次数与失败次数
3. 计算成功率 = 成功次数 / 总次数 * 100%

| 检查项 | 验收标准 | 实际结果 | 验收人 | 日期 | 通过 |

**测试方法：**
1. 启动批量任务（如批量评语生成 50 条）
