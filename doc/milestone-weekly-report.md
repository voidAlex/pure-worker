**最后更新：** YYYY-MM-DD
**所属里程碑：** M{1-5}
**报告周期：** 第 X 周（YYYY-MM-DD ~ YYYY-MM-DD）

---

## 使用说明

本文档用于每周 Sprint 的计划与验收记录，包含两个部分：

1. **§8.1 周计划模板**：每周初填写，明确目标、依赖、产出与风险
2. **§8.2 周验收模板**：每周末填写，记录完成情况、缺陷、质量与延期

填写规范：
- 任务 ID 必须与 `development-plan-v1.0.md` 中定义一致
- 状态使用：`todo` / `in_progress` / `blocked` / `done`
- 阻塞项必须填写原因、影响范围、临时方案、解除条件
- 所有日期使用 ISO 8601 格式（YYYY-MM-DD）
