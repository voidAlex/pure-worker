PureWorker

面向教师场景的本地优先桌面 AI 助手

🚧 开发中 - M1 工程底座阶段

PureWorker 是一款专为教师设计的 AI 辅助工具，采用本地优先架构，确保学生数据安全。通过智能化的教务管理、作业批改、家校沟通等功能，帮助教师提升工作效率，专注于教学本身。

🔒 本地优先

学生数据保存在本地，默认不上传云端，保护隐私安全

🤖 智能辅助

AI 生成评语、通知草稿，教师确认后生效，人审闭环

📋 教务管理

班级、学生档案、成绩、观察记录等一站式管理

💬 家校沟通

智能生成沟通内容，支持批量处理，提升沟通效率

Tauri 2.x Rust React TypeScript SQLite Tailwind CSS

📥 下载客户端

支持 Windows 和 macOS 平台
